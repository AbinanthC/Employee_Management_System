# Employee Management System

## Project Overview

This is a command-line Employee Management System developed using:

- Python
- PostgreSQL
- psycopg2

The project performs complete CRUD operations with database integration.

---

# Features Implemented

- Add Employee
- View Employees
- Search Employee by ID
- Update Employee Salary
- Delete Employee
- Search by Department
- Sort by Salary
- Export Employee Data to CSV

---

# Technologies Used

- Python
- PostgreSQL
- psycopg2
